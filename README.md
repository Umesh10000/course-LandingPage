# Eduvui Landing Page

A modern, fully responsive landing page built with raw HTML and Tailwind CSS. This project serves as a pixel-perfect recreation of the provided Figma design for an educational platform / coaching service.

## 🚀 Features

- **Fully Responsive:** Adapts beautifully from mobile devices all the way up to large desktop screens.
- **Tailwind CSS Utility-First Styling:** Clean, modern UI components relying entirely on Tailwind CSS utility classes via CDN. No external stylesheets needed.
- **Custom Gradients & Blends:** Features advanced UI touches including radiant background glows, mixed gradients, and dynamic layout overlapping to match the original design's aesthetic.
- **Modern Layouts:** Utilizes CSS Flexbox and CSS Grid layout patterns inside Tailwind frameworks for sections like "Top Categories" and "Featured Courses".

## 🛠️ Built With

- **HTML5** (Semantic markup)
- **Tailwind CSS** (via CDN for rapid prototyping & styling)
- **Google Fonts** (Inter & Poppins used for beautiful typography contrasts)

## 📂 Project Structure

- `index.html` - The single-page application markup and layout.
- `/Assets` - All images, SVGs, avatars, and visual assets used throughout the page.

## 💻 Running the Project Locally

Because this project uses raw HTML and Tailwind via CDN, there are no build steps or package installations required!

1. Clone or download this repository.
2. Navigate into the project folder.
3. Open `index.html` directly in your favorite web browser (or use VS Code's "Live Server" extension for hot reloading as you edit).

## 🎨 Design Highlights

- **Hero Section:** Features a dynamic radiant background gradient glow perfectly framing the subject asset.
- **Course Library Grid:** Clean hover animations and CSS grid layout adapting from 1 column to 3+ columns for larger screens.
- **Testimonial Slider UI:** Responsive review cards stacked to emulate an overlapping interactive carousel.
